# Smart Canteen 🍴

A modern canteen management system built with React.

## Features (so far)
- User-side UI (shop browsing, cart, offers, favorites, profile)
- Role-based login: `User` and `Shopkeeper`
- Shopkeeper panel for each shop (own menu, own orders, own shop status)
- Orders flow from user checkout to shopkeeper order management
- Responsive design

## Shopkeeper Credentials (default)
- Juice Corner: `juice_admin / juice123`
- Tiffen Center: `tiffen_admin / tiffen123`
- Biriyani Point: `biriyani_admin / biriyani123`
- Bakery: `bakery_admin / bakery123`
- Tea Time: `tea_admin / tea123`
